# discussion-continuity

Save and continue discussions across sessions in Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| `continue-discussion` | Save discussion state for continuation in a future session |

## Commands

| Command | Description |
|---------|-------------|
| `/continue-discussion` | Invoke the continue-discussion skill |

## Installation

Add to your `.claude/settings.json`:

```json
{
  "plugins": [
    "/path/to/discussion-continuity"
  ]
}
```
