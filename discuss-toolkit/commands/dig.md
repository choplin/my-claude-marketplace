# dig - Intent Clarification

Start a structured interview to clarify intent before proceeding.

Load the dig skill and follow its process. Use AskUserQuestion repeatedly until:
1. Motivation is documented
2. Concrete examples are collected
3. Next action is determined
4. All claims are user-verified

Never fill gaps with assumptions. Present hypotheses for user confirmation.
