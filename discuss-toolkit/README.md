# discuss-toolkit

Discussion and idea exploration skills for Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| `discuss` | Deep discussion mode for exploring ideas without implementation |
| `name-project` | Suggest project names with conflict checking |
| `quick-chat` | Practical conversation with honest, constructive feedback |
| `understand-idea` | Clarify ideas before starting work |

## When Skills Activate

- **discuss**: "let's discuss", "explore this idea", "brainstorm"
- **name-project**: "name this project", "suggest a name"
- **quick-chat**: "what do you think", "A vs B", "which is better"
- **understand-idea**: "I have an idea", "help me understand", "before we start"

## Installation

Add to your `.claude/settings.json`:

```json
{
  "plugins": [
    "/path/to/discuss-toolkit"
  ]
}
```
